A simple terminal-based Tic-Tac-Toe game for two players, written in C++.

📝 Features
Two-player gameplay
Turn-based input
Board display in terminal
Win/draw detection

🛠️ Technologies Used
C++
Standard I/O (cin/cout)
Terminal/Console

Run game:
./tictactoe
